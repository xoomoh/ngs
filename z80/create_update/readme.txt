
Для сборки апдейта запускать create_update.bat

Файлы загрузчика (loader_ngs.rom) и основное ПЗУ NeoGS (neogs.rom)
сборщик берет из соотвествующих директорий, бинарик фпга для 
страницы 7 должен лежать в этой же директории под именем fpga.bin

Файл обновления под именем ngs_rom.upd должен лежать в корне 
SD карты, где его будет искать флешер.

Даты создания страниц прошивки задаются в bat файле.
