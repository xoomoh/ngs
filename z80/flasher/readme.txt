Run flasher from FLASHNGS.$C (Hobeta) or flasher.scl.

File ngs_rom.upd must be in root directory of SD card.

SD card must be installed to NeoGS.

============================================================
Флешер сделан в 2 видах: FLASHNGS.$C (Hobeta) и flasher.scl
Запускать какой удобнее и откуда удобнее.

Файл обновления под именем ngs_rom.upd должен лежать в корне 
SD карты, где его будет искать флешер.

SD карта должна быть установлена в NeoGS.
